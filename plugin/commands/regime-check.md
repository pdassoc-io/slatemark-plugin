Run a Regime Check using the connected Slatemark tools. Pull the
cross-asset backdrop into one read before taking any single-name view:

- the yield curve and real yields;
- credit spreads;
- the dollar;
- a financial-conditions read;
- realized vol and rolling correlations across the major assets over
  recent sessions.

Name the regime and ask where the dislocations are — the cross-asset
tension worth watching. Frame it as a read of the weather, not a
position: the question is what backdrop the user is trading in, never a
call to act.
